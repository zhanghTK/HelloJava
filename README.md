# Joolkit

Java toolkit.

---

## What's in it?

- `tk.zhangh.java.entity`
  - 使用注解生成基本SQL语句
- `tk.zhangh.java.io`
  - 基本文件，文件夹操作
  - 文件输入处理模板
- `tk.zhangh.log`
  - 日志切面
- `tk.zhangh.java.office.excel`
  - Excel解析
- `tk.zhangh.java.practice`
  - 对比反射API中isAccessible方法
  - 在构造方法内创建线程造成逸出
  - Map性能测试
  - 各种引用对象对比
  - 泛型通配符
  - 内省机制
  - 测试自动装箱，拆箱对反射的影响
  - 对比String和StringBuilder
  - try,catch,finally执行顺序
- `tk.zhangh.java.reflection`
  - 运行时类，对象信息分析
  - 方法运行时长统计
  - 反射帮助类
- `tk.zhangh.toolkit`
  - 签名，验证
- `tk.zhangh.java.x`
  - Socket Demo
    - 通信
    - Server
    - RPC
  - 生命周期框架
